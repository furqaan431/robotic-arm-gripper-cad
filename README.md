# robotic-arm-gripper-cad
Mechanical CAD model of a robotic gripper with assembly, renders, and 2D drawings
This repository contains the complete CAD design of a two-finger robotic arm gripper. The project includes parametric part models, full assembly, exploded views, and detailed drawings. Renders and STEP/IGES files are included to ensure compatibility with multiple CAD platforms. The design focuses on precision and synchronized jaw movement required in industrial automation.
